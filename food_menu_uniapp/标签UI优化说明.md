# 菜品标签 UI 优化说明

## 🎨 优化内容

### 1. 视觉效果升级

#### 渐变背景
- 使用线性渐变背景，增加层次感
- 三种颜色主题循环：红色系、绿色系、蓝色系
- 半透明效果，与主题色融合

#### 毛玻璃效果
- 添加 `backdrop-filter: blur(8px)` 实现毛玻璃质感
- 增强现代感和科技感

#### 边框光晕
- 彩色边框与标签主题色呼应
- 添加多层阴影，营造浮空效果
- 内阴影增加立体感

### 2. 动画效果

#### 浮动动画 (tagFloat)
- 标签轻微上下浮动
- 3秒循环，营造灵动感
- 缓动函数：ease-in-out

#### 光晕扫过动画 (tagShine)
- 45度角光晕扫过效果
- 3秒循环，增加炫酷感
- 使用伪元素实现

#### 图标弹跳动画 (tagIconBounce)
- Emoji 图标轻微缩放
- 2秒循环，增加趣味性
- 与标签浮动错开，避免单调

#### 点击反馈
- 点击时缩小至 95%
- 阴影减弱，模拟按下效果
- 使用 cubic-bezier 缓动函数

## 🚀 使用方法

### 数据格式
确保菜品数据包含 tags 字段：
```javascript
{
  id: 1,
  name: '宫保鸡丁',
  tags: '川菜,辣,下饭'  // 字符串，逗号分隔
  // 或
  tags: ['川菜', '辣', '下饭']  // 数组
}
```

### 自动解析
代码会自动将字符串格式的 tags 转换为数组：
```javascript
tags: item.tags && typeof item.tags === 'string' 
  ? item.tags.split(/[,，]/).filter(Boolean) 
  : []
```

### 添加新图标
在 `getTagIcon` 函数中添加映射：
```javascript
const iconMap = {
  '新标签': '🆕',
  // ...
}
```

## 🔧 自定义调整

### 调整动画速度
```scss
animation: tagFloat 3s ease-in-out infinite;  // 修改 3s
animation: tagShine 3s linear infinite;       // 修改 3s
animation: tagIconBounce 2s ease-in-out infinite;  // 修改 2s
```

### 调整标签大小
```scss
padding: 8rpx 16rpx;  // 修改内边距
font-size: 22rpx;     // 修改字体大小
border-radius: 20rpx; // 修改圆角
```

### 调整颜色主题
修改 `.tag:nth-child(3n+1/2/3)` 中的颜色值

### 禁用动画
注释掉 `animation` 属性即可

## 💡 最佳实践

1. **标签数量**：建议每个菜品 2-4 个标签
2. **标签长度**：建议每个标签 2-4 个字
3. **标签顺序**：重要标签放前面
4. **标签分类**：菜系 + 口味 + 特点的组合
