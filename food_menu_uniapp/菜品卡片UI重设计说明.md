# 菜品卡片 UI 重设计说明

## 🎨 设计理念

### 核心原则
- **卡片式设计**：采用现代卡片式布局，信息层次清晰
- **视觉焦点**：大图优先，突出菜品视觉吸引力
- **信息分层**：图片、标题、标签、描述、价格分层展示
- **交互友好**：清晰的操作区域，流畅的动画反馈

## 📐 布局结构

### 整体布局
```
┌─────────────────────────────┐
│                             │
│      菜品大图 (320rpx)       │
│   [分类标签]      [收藏按钮]  │
│                             │
├─────────────────────────────┤
│  菜品名称                    │
│  🌶️川菜  ⭐经典  🍚下饭      │
│  经典川菜，鸡肉鲜嫩...        │
│  ─────────────────────      │
│  ¥38 /份         [- 1 +]    │
└─────────────────────────────┘
```

### 区域划分

#### 1. 图片区域 (320rpx 高度)
- **全宽大图**：展示菜品视觉效果
- **渐变遮罩**：底部渐变，增强层次感
- **收藏按钮**：右上角，白色圆形按钮
- **分类标签**：左下角，半透明黑色标签

#### 2. 内容区域 (padding: 24rpx)
- **标题行**：菜品名称，32rpx 粗体
- **标签行**：最多显示 3 个标签
- **描述文本**：2 行截断，24rpx
- **操作栏**：价格 + 数量控制器

## 🎯 视觉设计

### 1. 卡片容器
```scss
- 圆角：24rpx
- 背景：毛玻璃效果 (backdrop-filter: blur(20px))
- 阴影：多层阴影，营造浮空感
- 边框：1px 半透明边框
- 动画：点击缩放 0.98
```

### 2. 图片区域
**大图展示**
- 宽度：100%
- 高度：320rpx
- 过渡：点击时放大 1.05 倍

**渐变遮罩**
- 位置：底部 120rpx
- 效果：从透明到半透明黑色
- 作用：保护底部标签可读性

**收藏按钮**
- 尺寸：64rpx × 64rpx
- 背景：白色 95% 不透明
- 图标：🤍 未收藏 / ❤️ 已收藏
- 动画：
  - 点击：缩放 0.88
  - 收藏：弹入动画 + 心跳动画

**分类标签**
- 位置：左下角 16rpx
- 背景：黑色 60% 不透明 + 毛玻璃
- 文字：白色 22rpx 粗体
- 圆角：20rpx

### 3. 内容区域

**标题**
- 字号：32rpx
- 字重：700 (粗体)
- 行高：1.3
- 截断：单行省略

**标签**
- 布局：横向排列，最多 3 个
- 尺寸：padding 6rpx 12rpx
- 圆角：16rpx
- 背景：渐变 + 毛玻璃
- 边框：1px 半透明
- 颜色：三色循环（红、绿、蓝）
- 字号：20rpx 粗体

**描述**
- 字号：24rpx
- 行高：1.6
- 颜色：次要文字色
- 截断：2 行省略

**操作栏**
- 布局：左右分布
- 顶部：1px 分隔线
- 内边距：16rpx 0

### 4. 价格展示
```scss
价格符号：¥ (24rpx, 粗体)
价格数值：38 (40rpx, 超粗体 800)
价格单位：/份 (22rpx, 中等字重)
颜色：#ff6b6b (统一红色)
```

### 5. 数量控制器
**减号按钮**
- 尺寸：56rpx × 56rpx
- 背景：输入框背景色
- 边框：2px 边框色
- 图标：− (28rpx)

**数量显示**
- 字号：28rpx
- 字重：700
- 宽度：36rpx 最小宽度

**加号按钮**
- 尺寸：56rpx × 56rpx
- 背景：红色渐变 (#ff6b6b → #ff8787)
- 阴影：红色光晕
- 图标：+ (28rpx 白色)
- 动画：点击缩放 0.85

## 🎬 动画效果

### 1. 卡片交互
```scss
点击卡片：
- transform: scale(0.98)
- 阴影减弱
- 图片放大 1.05

过渡时间：0.4s
缓动函数：cubic-bezier(0.34, 1.56, 0.64, 1)
```

### 2. 收藏动画
```scss
未收藏 → 已收藏：
1. 背景渐变：白色 → 红色渐变
2. 弹入动画：0.8 → 1.15 → 1.0
3. 心跳动画：旋转 + 缩放
4. 阴影增强：红色光晕

点击反馈：
- 缩放至 0.88
```

### 3. 按钮动画
```scss
加号/减号按钮：
- 点击缩放：0.85
- 过渡：0.3s cubic-bezier
```

## 🎨 颜色系统

### 主色调
- **价格红**：#ff6b6b (主要)
- **渐变红**：#ff6b6b → #ff8787 (按钮)

### 标签颜色（循环）
1. **红色系** (3n+1)
   - 背景：#ff6b6b10 → #ff6b6b18
   - 边框：#ff6b6b30
   - 文字：#ff6b6b

2. **绿色系** (3n+2)
   - 背景：#51cf6610 → #51cf6618
   - 边框：#51cf6630
   - 文字：#51cf66

3. **蓝色系** (3n+3)
   - 背景：#339af010 → #339af018
   - 边框：#339af030
   - 文字：#339af0

### 主题适配
- 卡片背景：`themeConfig.cardBg`
- 卡片边框：`themeConfig.cardBorder`

## 🎯 使用建议

### 1. 图片质量
- 建议尺寸：750×600 以上
- 格式：JPG/PNG/WebP
- 压缩：适度压缩，保证清晰度

### 2. 标签数量
- 建议：2-3 个标签
- 最多：3 个（自动截断）
- 顺序：重要标签在前

### 3. 描述文本
- 长度：40-60 字
- 内容：突出特色和口味
- 避免：过长或过短

### 4. 价格设置
- 格式：整数或一位小数
- 范围：建议 10-200 元
- 显示：自动格式化
