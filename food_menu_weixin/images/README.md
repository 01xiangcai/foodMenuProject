# TabBar 图标说明

此目录需要放置底部导航栏的图标文件。

## 需要的图标文件

1. **tab-home.png** - 首页未选中图标（建议尺寸：81x81px）
2. **tab-home-active.png** - 首页选中图标（建议尺寸：81x81px）
3. **tab-menu.png** - 点餐未选中图标（建议尺寸：81x81px）
4. **tab-menu-active.png** - 点餐选中图标（建议尺寸：81x81px）
5. **tab-profile.png** - 我的未选中图标（建议尺寸：81x81px）
6. **tab-profile-active.png** - 我的选中图标（建议尺寸：81x81px）

## 图标设计建议

- 未选中状态：灰色调，透明度较低
- 选中状态：使用主题色 #14b8ff（蓝色），可以添加发光效果
- 风格：简洁、现代、符合科技感主题

## 临时方案

如果暂时没有图标，可以使用在线工具生成，或者使用简单的几何图形作为占位符。
